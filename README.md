# Role-Based Access Control (RBAC) Dashboard

// Project Overview //

The Role-Based Access Control (RBAC) Dashboard is a web-based application built using React that provides a secure and user-friendly interface for managing users, roles, and permissions. This project simulates core RBAC functionalities, including dynamic permission assignment and role management, with a focus on usability, responsiveness, and modern design.


// Features //

1. User Management
Add, edit, and delete users.
Assign roles to users dynamically.
Manage user status (Active/Inactive).
2. Role Management
Add, edit, and delete roles.
Assign permissions (Read, Write, Delete) to roles dynamically.
3. Permissions
Clear and intuitive interface to manage role-based permissions.
Ensure roles are mapped correctly to permissions for secure access.
4. Responsive Design
Fully responsive UI that adapts to different screen sizes.
Clean and professional design for a seamless user experience.


// Tech Stack // 

Frontend: React, CSS
Styling: Custom CSS for a modern and responsive layout.
